Video-based driver fatigue / drowsiness detection

Classes:  
Alert / Awake  
Low Vigilance  
Drowsy

Approximate Size:  
Around 30 hours of RGB videos

Number of Subjects:  
60 subjects

Usage:  
Academic project / research use, subject to source/license terms

Why selected:  
Selected because it is video-based and supports temporal fatigue detection using models like CNN-LSTM, CNN-GRU, TCN, or lightweight 3D CNN.
